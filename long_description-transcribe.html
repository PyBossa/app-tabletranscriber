O Table Transcriber Transcribe é uma aplicação para o pybossa que visa a transcrição de tabelas em documentos históricos do Brasil, escaneados pelo IPEA.

Esta aplicação utiliza imagens de tabelas contidas em documentos históricos digitalizados e necessita da transcrição dos dados contidos nessas tabelas. O objetivo final é conseguir que todas as tabelas contidas nesses documentos sejam disponibilizadas de forma editável.

O que precisamos que seja feito nessa tarefa é a transcrição dos dados contidos nas tabelas.

A aplicação possui a seguintes ações:

  • Salvar transcrição incompleta
  • Salvar transcrição completa
  • Não há tabela
