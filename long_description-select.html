O Table Transcriber Select é uma aplicação para o pybossa que objetiva identificação de tabelas em documentos históricos do Brasil, escaneados pelo IPEA.

Esta aplicação utiliza imagens de documentos históricos digitalizados e visa a preparação dessas imagens para futuras tarefas. O objetivo final é conseguir que todas as tabelas contidas nesses documentos sejam disponibilizadas de forma editável.

O que precisamos que seja feito nessa tarefa é a identificação de tabela nas páginas dos livros digitalizados.

A aplicação possui a seguintes ações:

  • Sim
  • Não e
  • Não sei
